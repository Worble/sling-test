Install Elixer

Install Postgres

Install npm / yarn

This project assumes postgres username and password to be 'postgres'. You can change this in /api/config/dev.exs

Open /api/

Run 'mix local.hex'

Run 'mix deps.get'

Run 'mix phoenix.server'

Open /web/

Run 'npm i' or 'yarn'

Run 'npm start' or 'yarn start'
